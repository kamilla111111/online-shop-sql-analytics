# Online Shop SQL Analytics

Учебный SQL-проект интернет-магазина.

## Структура базы данных

База данных содержит 4 таблицы:

- customers — покупатели
- products — товары
- orders — заказы
- order_items — состав заказов

## Использованные SQL-конструкции

- INNER JOIN
- LEFT JOIN
- GROUP BY
- HAVING
- CTE (WITH)
- Window Functions
- RANK()
- Агрегатные функции

## Аналитические запросы

### Топ-5 клиентов по тратам
Определение покупателей с наибольшей суммой покупок.

### Самый популярный товар
По количеству проданных единиц.

### Товары без продаж
Поиск товаров, которые ни разу не заказывали.

### Средний чек
Расчёт среднего значения заказа.

### Самый дорогой заказ каждого клиента
Использование оконной функции RANK().

## Файлы проекта

- schema.sql — создание базы данных и таблиц
- insert_data.sql — тестовые данные
- queries.sql — аналитические запросы
